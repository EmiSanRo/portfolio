    Projects

    Linear Control project

    In this project I help in calculate the values of the PID controller for a system that was going to be used by the down-scaled system. MATLAB's System Identification were used to find the transfer function that described how the system works. PID Tuner was used to find the values of the PID controller. I also implemented the logic and connections required to integrate CAN bus communication between the different subsystems. In this case an ESP32 microcontroller wich were connected to a mobile hotspot that received the CAN messages. Using Arduino Cloud I created a dashboard that showed the values of the sensors and allowed to change the setup-point of the system remotely by sending back CAN messages. I also designed the dashboard and coded the logic.

    Refrigerator Project

    I welded the different components to the welding PCB board.Then programmed the part of the code where the sensor detected when the door were opened or a current peak where reached and the logic after that. When this happened a timer started and if the door was open for more than 1 minute the alarm would start in the case of an open door. In the case of a current peak the alarm would start immediatly. I also implemented the bluetooth connection between the refrigerator and a phone so that the user could receive a message when one of these conditions were met.

    Piano Project

    In this project I designed the whole piano using the software Processing. Also I programmed the piano using the same software so that the keys changed its state and play a note when the corresponding key was pressed. Also implemented the serial communication between the piano and the fpga board so that externals buttons can be used to play the notes.
